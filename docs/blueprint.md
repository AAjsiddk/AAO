# **App Name**: Najat Links

## Core Features:

- Link Aggregation: Centralized location for all Najat resources, including direct links to each resource with descriptive text and button.
- Day/Night Mode: Allow users to switch between a light and dark theme for optimal viewing experience.
- دفتر رمضان Link: Provide a direct link to download دفتر رمضان – دليلك العملي لشهر مختلف, hosted on Google Drive.
- Lumin PDF Link: Provide a direct link to download the Lumin PDF application from the Google Play Store.
- نجاتك بيدك Website Link: Provide a direct link to the نجاتك بيدك website.
- نجاتك بيدك App (Android) Link: Provide a direct link to download the نجاتك بيدك Android application from MediaFire.
- Telegram Bot Link: Provide a direct link to the Telegram bot for communication and updates.
- WhatsApp Channel Link: Provide a direct link to the WhatsApp channel for important notifications.

## Style Guidelines:

- Background color: A very light desaturated green (#F0F4F0) to create a calming and natural feel.
- Primary color: A soft green (#8FBC8F), offering a sense of growth and tranquility, making it versatile for different elements.
- Accent color: A muted olive (#A8B878) for contrast in buttons and interactive elements, staying within the analogous scheme.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Clean, intuitive layout with a focus on easy navigation. Each link should be clearly presented with a descriptive title and a prominent button.
- Use simple, elegant icons to represent each link type (e.g., a book icon for دفتر رمضان, a phone icon for the app link).  Consider cultural appropriateness and symbolism.
- Subtle hover effects on the buttons and a smooth transition between day and night modes.
